## Objective

The SOC Detection Lab project establishes a controlled, enterprise-style environment to **simulate cyber attacks, generate telemetry, and detect malicious activity using Splunk SIEM**. The lab ingests logs from Windows and Linux endpoints, correlates events generated by adversary emulation (Kali Linux) and Atomic Red Team, and produces detections aligned to **MITRE ATT&CK**. The goal is to demonstrate practical SOC, blue team, and detection engineering skills through hands-on implementation and clear documentation.

---

### Skills Learned

* Practical SIEM deployment and administration (Splunk Enterprise)
* Log ingestion, normalization, and correlation across multiple data sources
* Windows telemetry analysis using Sysmon and Event Logs
* Linux authentication and audit log analysis
* Adversary emulation and attack simulation (Kali Linux, Atomic Red Team)
* Detection engineering mapped to MITRE ATT&CK techniques
* Incident triage, investigation, and reporting
* Security documentation and GitHub project presentation
